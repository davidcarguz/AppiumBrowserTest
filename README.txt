Pasos Previos:
1. Intentar correr el Proyecto.
2. Si el proyecto falla, validar si la razon del problema es que el driver y la versión de chrome no son compatibles.
3. verificar la versión de chrome en el dispositivo.
4.descargar el chromedriver que se ajuste a la versión del dispositivo.
5. reemplaza el chromedriver en la siguiente ubicación C:\Program Files\Appium\resources\app\node_modules\appium\node_modules\appium-chromedriver\chromedriver\win
si selecciono una ubicación diferente para instalar appium debe referirse a esa carpeta.
6. el proyecto debe funcionar correctamente.
